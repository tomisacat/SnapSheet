# SnapSheet 1.0.0

**SnapSheet** is a pure SwiftUI bottom sheet for iOS 18+. It composes a full-screen background with a draggable, snap-to-detent floating panel — the map + sheet pattern you see in Apple Maps — with no UIKit bridges or system sheet presentation controllers.

## Requirements

- iOS 18+
- Swift 6.2+
- Xcode 26+

## Installation

Add SnapSheet via Swift Package Manager:

```swift
dependencies: [
    .package(url: "https://github.com/tomisacat/SnapSheet.git", from: "1.0.0"),
],
targets: [
    .target(
        name: "<YourAppTarget>",
        dependencies: ["SnapSheet"]
    ),
]
```

## Highlights

### Pure SwiftUI sheet with snap detents

Three resting positions — **collapsed**, **half**, and **expanded** — with velocity-aware snapping driven by `DragGesture` predicted end translation.

### Scroll handoff

When expanded, `ScrollView` content scrolls normally. A downward drag at scroll top hands off to sheet dragging so the sheet collapses naturally instead of fighting the scroll view.

### Composable API

- `**BackgroundSheetScene`** — all-in-one integration for background + sheet
- `**DraggableSheet**` — lower-level sheet view when you manage layout yourself
- `**SheetStateModel**` — `@Observable` state you can share across your UI
- `**SheetStyle**` — corner radius, handle size, and content padding

### Gesture coordination

`GestureOwner` tracks whether the background, sheet drag, or sheet scroll is active, keeping interactions predictable during drag and scroll transitions.
